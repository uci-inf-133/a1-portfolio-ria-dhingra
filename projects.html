ZipTrip (IrvineHacks 2025)

Description

ZipTrip is an AI-powered Travel Itinerary Generator which aims to help users plan their future vacations in an optimized and effective manner. Users can search for activities near their location and select their interests from curated suggestions. After inputting, they will receive an AI-generated detailed itinerary, complete with time slots and proximity optimization. Users can also save their plans as a PDF for easy sharing and offline access.

Tools & Technologies

  • Programming Languages: HTML/CSS
  • Framework: React.js
  • API or Platforms: Google's Gemini API, Mapbox API
